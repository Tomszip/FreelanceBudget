# Freelance Budget App

Esta aplicación permite calcular de forma rápida el presupuesto de un proyecto freelance en base a distintos parámetros.

## ¿Qué hace la app?

El usuario puede:

* Ingresar el nombre del proyecto
* Definir el valor por hora
* Seleccionar el nivel de complejidad (baja, media o alta)
* Indicar si el proyecto es urgente

Con estos datos, la aplicación calcula automáticamente el costo total del proyecto.

## Funcionalidades principales

* Pantalla de bienvenida
* Formulario de carga de datos
* Validaciones (evita datos incorrectos)
* Cálculo automático del presupuesto
* Pantalla de resultados
* Modo oscuro (se guarda aunque se cierre la app)
